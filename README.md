# Trabalho sore Git 

Este repositorio foi criado para entregar a lista de exercicios de desnvolvimento de sitemas. O objetivo foi resolver problemas com Git.
Cada exercicio foi feito em uma branch separada e depois unido na branch principal.

## Tecnologias

- HTML para a estrutura
- CSS para organizar o visual
- JavaScript para os calculos e a logica
- Git e GitHub para salvar e organizar o codigo
- GitHub CLI para gerenciar os pull requests pelo terminal

## Lista de Exercicios Concluidos

1. Soma Simples: Adicao de dois valores informados pelo usuario.
2. Calculo de Lucro: Subtracao para descobrir a diferenca entre venda e custo.
3. Desconto de 10%: Aplicacao de desconto em produtos de eletronicos.
4. Juros Simples: Calculo de emprestimo com taxa fixa de 5%.
5. Comissao: Calculo de salario fixo mais 5% sobre as vendas.
6. Media de Vendas: Soma e divisao das vendas de 7 dias da semana.
7. Estoque: Controle de entrada e saida com aviso caso o produto acabe.
8. Aumento de Preco: Reajuste de 15% no valor dos produtos.
9. Parcelas: Divisao do valor total em ate 5 vezes sem juros.
10. Juros Compostos: Calculo de rendimento de investimento ao longo do tempo.

## Como foi organizado o Git

Para manter o projeto organizado, usei o seguinte esquema:
1. Criei branches para cada problema (ex: feature-problem1).
2. Fiz o commit e o push de cada uma.
3. Usei o GitHub CLI para abrir pull requests para a branch dev.
4. No final, juntei a branch dev com a main para a entrega final.
